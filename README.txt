---- EN VERSION ----

Program

- This is a Python application that models an anthill with a maximum of 4 nests and 4 resources.
- Nests can contain up to 200 ants and resources can have a maximum quantity of 500.
- You can add obstacles by clicking directly on the screen.

To launch the application, you have to make the command:
python3 main.py

Buttons:
- Add a nest: Add a nest if the simulation is not started or if the maximum number of nests is not exceeded.
- Add a resource: Adding a resource if the simulation is not started or if the maximum number of resources is not exceeded.
- Start: Start the simulation
- Stop: Stop the simulation
- Finish: End the simulation, no more obstacles can be added.
- Reset: After the end of the simulation, the simulation can be reset.
- +/- speed: Adjust the speed

---- FR VERSION ----

Programme

- Ceci est une application Python qui modélise une fourmilière avec au maximum 4 nids et 4 ressources.
- Les nids peuvent contenir jusqu'à 200 fourmis et les ressources peuvent avoir une quantité de 500 au maximum.
- On peut ajouter des obstacles en cliquant directement sur l'écran.

Commandes
Pour lancer l'application il faut faire la commande:
python3 main.py

Boutons:
- Add a nest : Ajout d'un nid si la simulation n'est pas lancé ou si le nombre de nids maximal n'est pas dépassé.
-  Add a resource : Ajout d'une ressource si la simulation n'est pas lancé ou si le nombre de ressources maximal n'est pas dépassé.
- Start: Lancer la simulation
- Stop : Stop la simulation
- Finish: Termine la simulation, on ne peut plus ajouter d'obstacles.
- Reset: Après la fin de la simulation, on peut remettre à zéro la simulation.
- +/- vitesse :  Ajuster la vitesse
